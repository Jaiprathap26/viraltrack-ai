## What is ViralTrack.ai?
AI-powered virality scoring for TikTok, Instagram Reels, and YouTube Shorts.
Know if your content will go viral BEFORE you waste time posting it.

## Who is it for?
- Content creators on TikTok/Reels/YouTube Shorts
- Social media managers
- D2C brands running UGC campaigns
- Influencer marketers

## Tech Stack
- **Frontend:** Next.js 14 + TypeScript + TailwindCSS
- **AI:** Claude claude-opus-4-5 for virality analysis
- **Payments:** Stripe (Free, Creator $15, Agency $49/mo)
- **Deploy:** Vercel

## Quick Start
```bash
git clone https://github.com/Jaiprathap26/viraltrack-ai.git
cd viraltrack-ai
npm install
cp .env.example .env.local
npm run dev
```
